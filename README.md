# Huudy

**Stream overlay powered by computer vision — no mods, no APIs, pure pixel reading.**

Huudy reads your game's HUD in real time and serves beautiful, customizable overlays directly to your streaming software.

## Quick Start

1. Run the installer and launch **Huudy**
2. Select your game from the dropdown
3. Point your capture source at your game window
4. Add a **Browser Source** in OBS/Streamlabs/Aitum pointing to `http://localhost:{port}`
5. Resize and style your overlay in the Huudy editor

Full setup guide → [huudy.app/setup](https://huudy.app/setup)
